# Capability: google-direct-oauth

> **Status:** REMOVED — superseded by the `clerk-oauth` capability at
> `openspec/specs/clerk-oauth/spec.md`. All direct-Google-OAuth flow,
> Auth0/Auth0-style SDK, and Direct-Google-PKCE code paths are gone.
> Kept on disk as a tombstone so older change references resolve.

## Purpose

This capability has been retired. The system uses Clerk for OAuth (Google) and never speaks to Google token endpoints directly. There is no remaining behaviour to express in requirements.

## Requirements

*None.* All prior requirements have been removed in
`openspec/changes/swap-google-direct-oauth-to-clerk/specs/google-direct-oauth/spec.md`.
