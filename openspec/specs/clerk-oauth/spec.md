# Capability: clerk-oauth

> **Status:** REMOVED — superseded by the `appwrite-auth` capability at
> `openspec/specs/appwrite-auth/spec.md`. All Clerk-backed auth flow
> requirements are retired.

## Purpose

This capability has been retired. The app no longer uses Clerk for
authentication; Appwrite is the active backend for login, session handling,
and token issuance.

## Requirements

*None.* All prior requirements were removed in
`openspec/changes/wipe-auth0-use-appwrite/specs/clerk-oauth/spec.md`.
